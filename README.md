The all-in-one vote management plugin for Minecraft servers.

## Features

- **Built-in NuVotifier Receiver** — No separate Votifier plugin needed
- **Multi-Platform** — Paper, Spigot, Folia, BungeeCord, Velocity
- **Vote Shop** — Players spend vote points on configurable items
- **Vote GUI** — Chest GUI with per-site vote buttons and status
- **Vote Streaks** — Daily streak tracking with milestone rewards
- **VoteParty** — Server-wide party when a vote goal is reached
- **Top Voters** — Leaderboards (daily, weekly, monthly, all-time)
- **Reward System** — Per-site, streak-based, milestone, and random rewards
- **Bedrock Support** — Native Bedrock Forms via Floodgate
- **Proxy Support** — BungeeCord and Velocity vote forwarding
- **12+ Integrations** — Vault, LuckPerms, PlaceholderAPI, CrazyCrates, and more
- **Multi-Language** — English and German included, fully customizable
- **Discord Webhook** — Vote logging to Discord channels
- **MySQL & SQLite** — Choose your database backend
